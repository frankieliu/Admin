Yes, you can cycle through images in a directory on macOS using Quick Look (Spacebar) for a quick preview, or by opening multiple images in Preview with Thumbnails in the sidebar for more control, but the most direct way is often Quick Look (Spacebar) after selecting all files in Finder. [1, 2, 3, 4]  
Using Quick Look (Easiest Method) 

1. Open the folder in Finder. 
2. Select all images (Command-A) or just one. 
3. Press the Spacebar to open Quick Look. 
4. Use the Up/Down Arrow keys (or Left/Right arrows) to cycle through all selected images. 
5. Press Escape (Esc) to close it. [1, 2, 3, 4, 5, 6]  

Using the Preview App 

1. Open the folder in Finder. 
2. Select all images (Command-A). 
3. Open them (Command-O or drag to Preview icon). 
4. In Preview, go to View &gt; Thumbnails (or ) to see them in the sidebar. 
5. Use the Up/Down Arrow keys to navigate through the thumbnails, and the main window will update. [3, 7, 8]  

Using Finder Gallery View 

1. Open the folder in Finder. 
2. Press Command-4 to switch to Gallery View. 
3. Click the first image, then use the Arrow keys to browse. [1]
