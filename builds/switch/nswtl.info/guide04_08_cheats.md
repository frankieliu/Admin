Using cheats in EdiZon
You can find most of the answers to typical questions in the FAQ section.
If something goes wrong, don't panic! First, read the Problems and Solutions section.
If you are a MacOS user, use these tips to avoid memory card problems.
Make backups of your saves as often as possible!
If the console freezes or shows no signs of life, turn it off by holding the (POWER) button for 30 seconds, then turn it on and forward the payload!
Preparatory work
Refresh kefir ▼ follow the instructions from the repository if you haven't done this before
Launch the Kefir Updater application via hbl
Go to the “Download Cheats” tab and download cheats from any source
The Internet must be connected and working.
One source may contain cheats that are not available in another, so keep this in mind.
CheatSlips.com requires authorization!
Wait until the cheats are downloaded and unpacked.
Press (+) to close the program
Return to the main menu
Working with cheats
Launch the game you want to use the cheat for.
Simultaneously press L + Down Arrow + R3 (or L + R + Up Arrow if you are using 4IFIR) to open the Tesla menu
Select “ EdiZon ” -> “ Cheats ”
By default all cheats are disabled, enable the ones you need.
After restarting the game, the cheats you activated earlier will also be active!
To avoid loading unnecessary cheats, disable all existing cheats before loading a game save, and then, after the save is loaded, activate only the ones you need.
Close Tesla with the (B) button
To disable the cheat, go to Tesla -> “ EdiZon ” -> “ Cheats ” again and disable the cheat.
