Using FTP
You can find most of the answers to typical questions in the FAQ section.
If something goes wrong, don't panic! First, read the Problems and Solutions section.
If you are a MacOS user, use these tips to avoid memory card problems.
Make backups of your saves as often as possible!
If the console freezes or shows no signs of life, turn it off by holding the (POWER) button for 30 seconds, then turn it on and forward the payload!
This program is needed to upload files to the console without removing the memory card. Download speeds will be significantly slower, but you won't have to restart the console and, consequently, the firmware. ▼

What you will need
Launched CFW ▼
Homebrew Launcher configured and working
Latest version of ftpd ( NRO )
Instructions
Part I - Installing the Program
Turn off your Switch and insert its memory card into your PC.
Copy  .nro  the ftpd file to the folder  switch  in the root of the console's memory card.
Insert the memory card into the console and run CFW ▼
Part II - Connecting to an FTP Server
Your IP address will be shown on the console screen in the top line.

Connect to the server with the address  указанный-ip:5000  in any way convenient for you.
