## Factory Reset Method

### Stock Console Reset

1. **System Settings**:
   - Go to System Settings
   - Scroll down to System
   - Select "Formatting Options"

2. **Choose Reset Type**:
   - "Format console memory"
   - "Delete all save data"
   - "Restore factory settings"

3. **Confirm Reset**:
   - Read warnings carefully
   - Enter PIN if set
   - Confirm reset choice
   - Wait for completion

### After Reset

1. **Initial Setup**:
   - Language selection
   - Region settings
   - Time and date
   - WiFi configuration

2. **System Update**:

3. **Force Reset**:
   - Hold POWER for 30 seconds
   - Try reset again
   - Contact support if persists

1. **Wait Patiently**:

   - Don't interrupt

2. **Force Restart**:
   - Hold POWER 15 seconds
   - Try reset again
