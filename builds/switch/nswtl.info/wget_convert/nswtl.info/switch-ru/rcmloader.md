RCM Loaders are convenient devices for injecting payloads into your Nintendo Switch without needing a computer. They make hacking more portable and user-friendly.

## What is RCM?

RCM (Recovery Mode) is the Nintendo Switch's recovery mode that allows payload injection through USB connection. RCM loaders automate this process.

## Types of RCM Loaders

### 1. RCM Jig + PC/Phone

**Traditional Method**:
- Jig to enter RCM
- Computer or phone for injection
- Software-based payload injection
- Most flexible option

### 2. RCM Loader Dongles

**All-in-One Solutions**:
- Combined jig and injector
- Battery-powered
- Portable and convenient
- Single button operation

### 3. Built-in RCM (Modchip)

**Permanent Solution**:
- Installed in console
- Auto-injects on boot
- Most convenient
- Requires installation

## Using RCM Loaders

### Traditional Method (Jig + Injector)

1. **Enter RCM Mode**:
   - Insert jig in right Joy-Con rail
   - Hold VOL+ and press POWER
   - Screen stays black

2. **Connect to Injector**:
   - Connect USB cable
   - Launch injection software
   - Select payload file

3. **Inject Payload**:
   - Click inject button
   - Console boots into payload
   - Remove USB cable

### RCM Loader Dongle Method

1. **Enter RCM Mode**:
   - Use built-in jig or external
   - Follow console instructions
   - Prepare for injection

2. **Connect RCM Loader**:
   - Connect to Switch USB-C
   - Wait for connection
   - LED indicates ready

3. **Inject Payload**:
   - Press button on dongle
   - Payload injects automatically
   - Console boots

### Built-in Modchip

1. **Power On Console**:
   - Modchip detects boot
   - Auto-enters RCM
   - Auto-injects payload

2. **Select Payload**:
   - May have boot menu
   - Choose desired payload
   - Boots automatically

## Troubleshooting

### Common Issues

**"No Response from Console"**:
- Check RCM mode entry
- Verify jig connection
- Try different USB cable
- Check USB port

**Payload Not Injecting**:
- Verify payload file
- Update injector software
- Try different payload
- Check device drivers

**RCM Loader Not Working**:
- Charge the device
- Check battery level
- Try different USB cable
- Contact support

### Console Issues

**Won't Enter RCM**:
- Console may be patched
- Jig not making contact
- Try different method
- Check serial number

**Black Screen After Injection**:
- Normal for first boot
- Wait 30-60 seconds
- Try different payload
- Check SD card files
