# What you will need
Ability to run payloads through Fusée Gelée
# Recommended method
## Installing AutoRCM
1. Log in to hekate ▼ using an exploit supported on your console, by running the payload  payload.bin  from the kefir  .zip  archive ▼
    1. Hold (VOL-) on the console when the kefir logo appears to enter hekate
        - Skip this step if you are already in hekate
2. Go to “ Tools ” -> “ Archive bit - AutoRCM ” (at the bottom of the screen) and set “ AutoRCM ” to “ ON ”
3. Press “ Home ” at the top of the screen
4. Run custom firmware ▼
## Disabling AutoRCM
1. Log in to hekate ▼ using an exploit supported on your console, by running the payload  payload.bin  from the kefir  .zip  archive ▼
   1. Hold (VOL-) on the console when the kefir logo appears to enter hekate
      - Skip this step if you are already in hekate
2. Go to “ Tools ” -> “ Archive bit - AutoRCM ” (at the bottom of the screen) and set “ AutoRCM ” to “ OFF ”
3. Press “ Home ” at the top of the screen
4. Run custom firmware ▼
