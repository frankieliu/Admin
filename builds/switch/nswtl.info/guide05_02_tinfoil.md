With the tinfoil app, games can be installed directly from a remote online server, which is constantly updated by the community. Essentially, the program is similar to FreeShop for the 3DS.

How does tinfoil work?
NSWTL CHAT
This program is not recommended for installing games from a PC. Use it exclusively for installing games over the internet.

Preparatory work
Turn off the console
Insert the memory card into the PC
Establish kefir ▼ , if you haven't done it yet
Insert the memory card into the console and run CFW ▼
Run DBI via HBL
Install  Tinfoil [050000BADDAD0000]  from folder  games 
Working with tinfoil
Launch tinfoil from the console's main menu
Wait for the list of games available for installation to load.
In the windows that appear, click OK
Errors may also appear, ignore them.
Go to the desired section, select the game and press (A)
The New Games section contains all the games that are on the server.
To search, press (+) on the Joy-Con while on the New Games line in the left side of the menu
The New Updates section contains updates for the games you have installed.
The New DLC section contains DLC for the games you have installed.
In the menu that appears, select “ Install all DLC ” and “ Install all updates ” so that the latest update and all available DLC are installed along with the game.
Click “ Install ” to begin the installation.
Press (B) to collapse the window that appears.
The download progress and speed will be displayed in the “ Download ” panel that appears at the top of the screen.
You can add games to your download list using the method described above. The entire list can be viewed in the " Queue " tab, which is located on the left tab bar.
After the selected game is downloaded, exit the program by pressing the (HOME) button
Do not enable the " Standard encryption " option! Games installed this way will not work!

After installing updates using tinfoil, we recommend going to DBI and selecting " Cleanup Orphaned Files ." This option will help remove junk files left behind by tinfoil during the update installation.

Possible problems and errors
I accidentally deleted game tickets in tinfoil
A: To reset tickets, run tinfoil , go to the “ Tickets ” tab, and for each game do the following:

In the game, press (Y)
Press (R)
Enable the “ Install Ticket ” option
Click “ Start Installation ”

SSL connection failed
You need to set the correct time and date on your console. Make sure the internet is enabled on your console, launch the program, and press (A). If time synchronization is disabled in the console settings, you will need to restart the program. After restarting, launch the program again. If the time on the console doesn't match your local time after synchronization, make sure the correct time zone is set in the console settings.
