# Existing hacking methods

Before you begin, enable display of file extensions if you are on Windows.

Currently, all Nintendo Switch consoles can be flashed. However, depending on the console's revision and the presence of vulnerabilities, flashing methods may vary. Therefore, the first step is to determine which method is right for you.

# Types of hacking
## Frozen Rocket
[fusee gelee](./guide03_01_fusee_gelee.md)
This hack can only be used on consoles released before the summer of 2018. Even though consoles released after that time were based on the same hardware platform, the vulnerability was patched. If you don't know whether your console is vulnerable to Fusée Gelée, below you'll find a tool for selecting the vulnerability type based on its serial number.

## Caffeine
[prep caffeine](./guide02_02_prep_caffeine.md)
This type of hack can be used to hack any Nintendo Switch with system software version 4.0.0 - 4.1.0. This is a software vulnerability tied to a specific system software version and only works on that version.

## Chipovka
[chip](./guide02_03_modchip.md)
Any Nintendo Switch can be hacked using chipping. Mariko chips ▼ It can only be hacked by chipping. First-revision consoles with a patched vulnerability and firmware higher than 4.1.0 can also only be hacked by chipping.

▼ The codename "Mariko" refers to the second-revision prefixes (the second letter in the serial number is a consonant, such as XK* or XJ*), as well as the Switch Lite and OLED. They are assembled using a new process technology, resulting in improved battery life (large switches). They can be flashed only by chipping.
